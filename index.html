Graduation Project ‚Äì Coins Sorting and Counting Machine

Designed and developed an automated Coins Machine that can sort, separate, and count coins based on their size and weight. The machine utilizes a combination of hardware and software components. The hardware includes a load cell (weight sensor), a motion sensor (PIR), and a sloped passage with precision-cut slots (made using a CNC machine) to separate coins by diameter. A DC motor with an unbalanced rotor provides vibration to guide the coins down the ramp. An LCD display shows real-time counting results.

The software was developed using Arduino Uno programmed in C language. The project went through multiple prototyping stages: one version relied on a motion sensor to detect coins, and another on weight measurements for enhanced accuracy and validation.
